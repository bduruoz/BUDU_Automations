4. Mini Kod Parçaları (Kopyala-Yapıştır)
LM-Studio çalışmazsa otomatik açtırma:
Python
Copy
# ai/lm_studio_launcher.py
import subprocess, psutil, time
def ensure_lm_studio(port=9090, exe=r"C:\Program Files\LM Studio\LM Studio.exe"):
    for p in psutil.process_iter(['name']):
        if "lm studio" in p.info['name'].lower(): return
    subprocess.Popen([exe]); time.sleep(8)
Entry script başına:
Python
Copy
from ai.lm_studio_launcher import ensure_lm_studio
ensure_lm_studio()
